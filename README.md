# Spider-Man VR — Ultimate Edition

A VR Spider-Man game with web swinging, all Insomniac villains, story missions, and NYC.

## Files

| File | Purpose |
|------|---------|
| `index.html` | Main HTML — all screens, CSS, loads scripts |
| `web.js` | Screen manager, lobby UI, character select, apartments, toast |
| `missions.js` | All missions: Story, Insomniac PS4/PS5, Amazing Spider-Man 2, Stealth, Boss Rush |
| `city.js` | NYC 3D city — skyscrapers, streets, apartments, Central Park, bridges, web anchor points |
| `swing.js` | Web swinging physics, player movement, game loop, combat, HUD |
| `villains.js` | All villains with 3D models: Kingpin, Electro, Vulture, Rhino, Doc Ock, Venom, Green Goblin, Carnage + more |
| `characters.js` | Spider-Man suit mesh, MJ (pregnant), Miles NPC, Ned, Uncle Ben, Uncle Aaron, Stan Lee |
| `vr.js` | WebXR VR: web shooter on right hand, locomotion on left stick, TV remote, haptics, VR HUD |

## Controls (Desktop)
- WASD — swing direction (air control)
- SPACE / LMB — shoot web
- E — release web
- F — jump / swing launch
- Q — web zip to roof
- Mouse drag — look around

## Controls (VR — Quest/Pico)
- Right trigger — shoot web (point hand at buildings!)
- Left stick — locomotion
- Right stick — snap turn
- Right grip — punch nearest enemy (haptic feedback!)
- Left grip — release web
- B button — jump boost
- Left controller — holds TV remote for mission select
